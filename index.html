Projeto: Calculadora de Médias

Esse projeto possui um formulário com 3 campos, ao preênche-los devidamente as informações são adicionadas em uma tabela com 3 colunas. Se o aluno tirar a nota abaixo da nota mínima um emoji junto com uma mensagem de reprovado aparecerá na tabela. Da mesma forma, se a nota for acima a mensagem aparecerá e o emoji será de felicidade.

Ferramentas usadas:

  • HTML 5
  • CSS 3
  • JavaScript
